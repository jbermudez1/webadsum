Plataforma para la gestión
de créditos financieros.

Todas sus operaciones desde desde la prospección y originaciónde los créditos, hasta la cobranza y recuperación de los mismos.

Beneficios:

  • Permite controlarbases de datos de prospectos de diferentes fuentes (Campañas, Expos, CallCenter, etc).
  • Personaliza todos los datos necesarios para la Solicitud de crédito.
  • Personaliza el registro de estudio socio económico por solicitud.
  • Diferentes plantillas de contrato, según sea el producto de financiamiento.
  • Integración de alertas de cobranza via e-mail y SMS.
  • Administra cuentas bancarias pesos / dólares.
  • Permite la integración de estados de cuenta de bancos para realizar.
    • conciliación bancaria.
  • Permite la generación de tablas de amortización.
  • Permite contabilidad multiOficinas.
  • Atiende requerimientos de SAT respecto a Contabilidad Electrónica.

Características:

  • MultiUsuarios.
  • Multi Sucursales.
  • Diseñado para funcionar desde la “nube” de internet.
  • Repositorio de expedientes digitales de operaciones de crédito.
  • Administra las diferentes fases relacionadas en la gestión de créditos:.
    • Promoción.
    • Solicitud de crédito.
    • Evaluación.
    • Autorización y Contratación.
    • Gestión de crédito.
    • Gestión de cobranza.
  • Permite personalización de:.
    • Campos personalizados de clientes.
    • Datos de solicitud de crédito.
    • Datos de Estudio Socio Económico.
    • Formato de contratos de crédito.
